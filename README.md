## この講座で学べること

* React + TypeScript + Web API を使った本格的な開発

* Web Socket を使った、リアルタイムのチャット機能
* ログイン／ユーザー登録などの認証機能などなど
* jotai というライブラリを使ったうグローバルステート管理についても学ぶことができます。
